## Features

- 🎮 **Player vs AI** — Smart AI opponent with adjustable difficulty
- 🌟 **Neon visuals** — Glowing paddles, ball trail, particle burst effects
- 🔊 **Sound effects** — Synthesized audio for hits, bounces and scoring
- ⚡ **3 Difficulty levels** — Easy, Medium, Hard
- ⏸️ **Pause / Resume** — Press `Space` or the Pause button
- 📱 **Mobile friendly** — Full touch support for phones and tablets
- 🏆 **Win condition** — First to 11 points wins the set
- 📊 **Post-game stats** — Shows total rally count after each match

---

## How to Play

| Control | Action |
|---|---|
| Move mouse up/down | Control your paddle |
| Touch and drag | Control your paddle (mobile) |
| `Space` | Pause / Resume |
| `Esc` | Back to main menu |

- You control the **blue paddle** on the left
- The **red paddle** on the right is the AI
- The ball speeds up slightly with every rally
- Hit the ball with the edge of your paddle for sharper angles

## Getting Started

### Play instantly
Just open `index.html` in any modern browser — no server or build step needed.

### Deploy to GitHub Pages
1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Click **Save**
5. Your game will be live at `https://YOUR-USERNAME.github.io/ping-pong-game/`
